# 查看顾客画像弹窗

**US ID**: US-03  
**所属分组**: profile  
**状态**: draft  
**创建日期**: 2026-02-25

---

## User Story

**作为** 导购  
**我想要** 查看顾客画像弹窗  
**以便于** 了解顾客偏好、尺码、历史购买等信息，提供个性化服务

---

## 验收标准（AC）

1. **AC-1**: 弹窗显示顾客基本信息（姓名、会员等级、积分、头像）
2. **AC-2**: 弹窗显示偏好信息（风格偏好、颜色偏好、价格偏好）
3. **AC-3**: 弹窗显示尺码信息（上衣、裤子、鞋子）
4. **AC-4**: 弹窗显示最近 3 次购买记录（商品、金额、日期）
5. **AC-5**: 弹窗显示生日信息（今日/近 7 天生日高亮提示）
6. **AC-6**: 如有投诉记录，弹窗显示投诉提示（脱敏）
7. **AC-7**: 弹窗支持滑动查看更多详情
8. **AC-8**: 弹窗加载时间 < 1 秒

---

## 约束与假设

**约束**:
- 敏感信息脱敏显示（如手机号中间 4 位）
- 投诉记录仅显示"有投诉记录"，不显示详情

**假设**:
- 顾客画像数据已存储在 CRM 系统
- 导购有权限查看顾客画像

---

## UI 要点

- 弹窗布局清晰，信息分层展示
- 重要信息（生日、偏好）用醒目颜色标注
- 支持快速关闭弹窗
